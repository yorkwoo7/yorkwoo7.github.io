Visual Component 4.7 Blender渲染插件安装

Visual Components（简称VC），是一款在智能制造、数字化工厂和数字孪生领域广泛使用的3D制造仿真软件。它以直观的界面和对多种CAD文件格式的支持，为用户提供了轻松设计和集成数字仿真环境的可能。

Blender，作为一款功能全面的免费开源3D创作套件，不仅支持建模、渲染、动画制作等3D管线的各个环节，而且在电影工业中也得到了广泛应用。

通过将VC与Blender结合，用户可以直接在Blender中操作和渲染VC的3D模型，实现更为逼真的视觉效果。以下是渲染前后的对比图：

气相/汽相之争

（未询问讨论人的个人意见，文章中人物身份均做消隐处理）

今天(24-04-30)大家为气相/汽相争论不休了。

起因是昨天（4-29号）EMI电子制造产业学院公众号发布了《真空气相焊专项精修训练营报名通知》。

业内人士指出，错了，应该是真空汽相焊才对。于是引发了一场讨论。有点意思。

然后院领导指示进行修改。

然后就开始了一场讨论。

首先本人观点自然认为：气是对的。理由是：气被加在“相” 字之前，所以应该确指是物质形态，所以应该用”气“。

我的其他理由还有各类硬件厂商的习惯性用法。比如Rehm，顶尖再流设备厂商之一。去年在写通孔回流焊团体标准的时候，就已经注意到这个术语的不一，和REHM袁工等工程师已经有过讨论，所谓”研究过“。

比如：苏威，全球最大的气相液供应商之一，Galden是他们的标志性产品。

比如：浩宝，国产真空气相焊厂商，他们的新闻稿中用的也是”气“。

没过一小时，我的BOSS表示应该修订，理由很强力，各类标准文件中，用的都是“汽”。

不过因为公众号文章只能修改一次，20个字符，此前已经被修改过，因此没有办法再改了。

大佬们中间在持续讨论，表达各自意见。

于是加大了讨论范围，和业界大佬、专家、老师们一起。

内网远程桌面连接卡在加密很久的解决

发表于 2023-07-21 更新于 2024-04-06 分类于 IT

内网远程桌面长时间加密远程连接的解决

内网使用远程桌面时，一直提示“正在加密远程连接…”，很久才能连接上。产生的原因是内网没有互联网连接，无法连接到证书服务器，一直等到连接超时，才继续执行。

解决办法是关闭证书更新。

1、键入组合键Win+R，输入gpedit.msc，回车进入组策略管理器

2、选择“计算机配置”>”管理模板”

3、选择“系统”>“internet通信管理”>选择”Internet通信设置“

4、双击”关闭根证书更新“，将值改为”已启用“

5、键入组合键Win+R，输入gpupdate /force 回车，更新组策略或者重新启动操作系统，生效。

第三方软件看到Windows映射盘的解决

有时候，我们在CMD使用net use命令或者在资源管理器映射网络驱动器之后，第三方软件却无法看到和使用这个磁盘。

处理办法如下：

1打开“程序和功能”里的“启用或关闭Windows功能”，“SMB直通”前打上勾

2注册表创建EnableLinkedConnections

详细步骤：

  1. 单击“开始”，在“运行”框中键入regedit，然后按Enter。

  2. 找到并右键单击注册表子项HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Policies\System

  3. 指向“新建”，然后单击“DWORD值”。

  4. 键入EnableLinkedConnections，然后按Enter。

  5. 右键单击“EnableLinkedConnections”，然后单击“修改”。

  6. 在“数值数据”框中，键入1，然后单击“确定”。

  7. 退出注册表编辑器，然后重新启动计算机

修复Windows回收站

没有注意，Windows 7的回收站经过VMWare OS optimized tool优化过后，回收站不可用，文件删除时被强制永久删除，并且回收站的配置属性变为灰色，无法修改。OS optimized tool的Rollback还没有这个选项的回复，怎么办？

将以下内容保存为xxx.reg，双击导入注册表，重启即可。

Windows Registry Editor Version 5.00

[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Policies\Explorer]
“NoRecycleFiles”=dword:00000000
“ConfirmFileDelete”=dword:00000000
